2. เข้าสู่ระบบ

login.html — เปิดหน้าแรกที่ต้องทำก่อนใช้งานทุกครั้ง

Admin ครูผู้ช่วย ดูอย่างเดียว
วิธีเข้าสู่ระบบ
  1. เปิดเว็บไซต์ศูนย์ เช่น https://kumon-qr.vercel.app/login.html
  2. กรอก อีเมล ที่ได้รับจากผู้ดูแลระบบ
  3. กรอก รหัสผ่าน (อย่างน้อย 6 ตัวอักษร)
  4. ทำเครื่องหมาย "จำการเข้าสู่ระบบ" หากใช้อุปกรณ์ส่วนตัว
  5. กดปุ่ม "เข้าสู่ระบบ" — ระบบจะพาไปหน้าหลักโดยอัตโนมัติ
ออกจากระบบ
  • กดปุ่ม "ออกจากระบบ" หรือ "ออก" ที่มุมบนขวาของทุกหน้า
  • ระบบจะล้าง Session และกลับมาหน้า Login ทันที
⚠️ ความปลอดภัย: หากใช้คอมพิวเตอร์สาธารณะ ควรออกจากระบบทุกครั้งหลังใช้งาน และไม่ควรติ๊ก "จำการเข้าสู่ระบบ"

4. Dashboard — สถานะนักเรียน

dashboard.html — ดูสถานะนักเรียนแบบ Realtime และเขียนบันทึก

Admin ครูผู้ช่วย ดูอย่างเดียว
สรุปสถิติด้านบน (4 ช่อง)
  • นักเรียนทั้งหมด — จำนวน active ทั้งหมดในระบบ
  • กำลังเรียน — มีรายการ check-in ที่ยังไม่ออก
  • ว่าง/ยังไม่เข้า — ยังไม่มีรายการวันนี้ หรือ check-out แล้ว
  • รายการเปิดอยู่ — จำนวน session ที่ยังค้างอยู่ (อาจ > นักเรียน ถ้าเรียนหลายวิชา)
ตัวกรองนักเรียน
  • ทั้งหมด / กำลังเรียน / ว่าง — กรองตามสถานะ
  • ทุกวิชา / MATH / THAI / ENG — กรองตามวิชาที่ลงทะเบียน
  • สามารถใช้ตัวกรองทั้งสองพร้อมกันได้ เช่น ดูเฉพาะ "กำลังเรียน" วิชา "MATH"
การ์ดนักเรียนแต่ละใบแสดง
  • ชื่อ-ชื่อเล่น และสถานะ (สีส้ม = กำลังเรียน / สีเขียว = ว่าง)
  • แถบวิชาที่ลงทะเบียน (MATH/THAI/ENG) และระดับชั้น
  • รายละเอียด session ปัจจุบัน: วิชา เวลาเข้า ระยะเวลา
  • จำนวนบันทึกวันนี้ (📝 X บันทึกวันนี้) — อัปเดตอัตโนมัติ
เขียนบันทึกพัฒนาการจาก Dashboard (Admin / ครูผู้ช่วย)
  1. กดปุ่ม "✏️ บันทึก" บนการ์ดนักเรียนที่ต้องการ
  2. พิมพ์บันทึกในช่องที่ปรากฏ (กด Ctrl+Enter เพื่อบันทึกเร็ว)
  3. กดปุ่ม "บันทึก" — ระบบจะอัปเดตจำนวนบันทึกบนการ์ดทันที
Realtime: หน้า Dashboard อัปเดตสถานะโดยอัตโนมัติเมื่อมีการสแกน QR ไม่ต้องรีโหลดหน้า
📷

5. สแกน QR — บันทึกเข้า/ออก

scan.html — บันทึกเวลาเข้าและออกห้องเรียนด้วยกล้องมือถือ

Admin ครูผู้ช่วย
วิธีสแกน QR
  1. กดปุ่ม "เริ่มสแกน" — เบราว์เซอร์จะขอสิทธิ์กล้อง ให้กด อนุญาต
  2. จ่อ QR Code ของนักเรียนให้อยู่ในกรอบกล้อง
  3. เมื่อสแกนสำเร็จ ระบบจะแสดงผล:
      — เข้าเรียน หากยังไม่มี session เปิดอยู่
      — ออกจากห้องเรียน + ระยะเวลา หากมี session ค้างอยู่
  4. กดปุ่ม "หยุด" เมื่อเสร็จสิ้นการสแกน
ตรวจสอบประวัติล่าสุด
  • รายการสแกนล่าสุด 10 รายการแสดงที่ด้านล่างของหน้า
  • แสดง ชื่อ · วิชา · สถานะ (เข้า/ออก) · เวลา
บันทึกแบบ Manual (ทดสอบ)
  • กรอก Payload ในรูปแบบ qr_code:subject_id แล้วกดปุ่ม "บันทึก"
  • ใช้สำหรับทดสอบโดยไม่ต้องใช้กล้อง
⚠️ ป้องกันสแกนซ้ำ: QR เดิมจะถูกล็อคไว้ 8 วินาที เพื่อป้องกันการบันทึกซ้ำโดยไม่ตั้งใจ
👨‍🎓

6. จัดการนักเรียน

admin.html — เพิ่ม แก้ไข ค้นหา และสร้าง QR Code นักเรียน

Admin ครูผู้ช่วย
เพิ่มนักเรียนใหม่
  1. กดปุ่ม "+ เพิ่มนักเรียน" ด้านบนขวา
  2. กรอกข้อมูล: ชื่อ, ชื่อเล่น, ระดับชั้น
  3. เลือกวิชาที่ลงทะเบียน: MATH / THAI / ENG (เลือกได้หลายวิชา)
  4. กดปุ่ม "บันทึก" — ระบบจะสร้าง QR Code อัตโนมัติ
ค้นหาและกรองนักเรียน
  • พิมพ์ในช่องค้นหา: รองรับ ชื่อ, ชื่อเล่น, ระดับชั้น, วิชา
  • กดปุ่มวิชา MATH / THAI / ENG เพื่อกรองตามวิชา
สร้างและพิมพ์ QR Code
  1. กดปุ่ม "📱 สร้าง QR ทั้งหมด" เพื่อสร้าง QR ของนักเรียนทุกคนพร้อมกัน
  2. หรือกดปุ่ม QR บนการ์ดนักเรียนแต่ละคนเพื่อสร้างเฉพาะราย
  3. กดปุ่ม "พิมพ์ QR" เพื่อเปิดหน้าต่างพิมพ์ของเบราว์เซอร์
  4. กดปุ่ม "ซ่อน" เพื่อซ่อน QR Code หลังพิมพ์เสร็จ
QR Code แต่ละใบ ใช้ได้กับวิชาที่นักเรียนลงทะเบียนเท่านั้น หากนักเรียนเรียน 3 วิชา จะมี QR 3 ใบ (MATH, THAI, ENG)
🔔

7. การแจ้งเตือน

notify.html — ตั้งค่านักเรียนที่ติดตามและรับแจ้งเตือนเมื่อสแกน QR

Admin ครูผู้ช่วย
เปิดรับการแจ้งเตือนของเบราว์เซอร์
  1. กดปุ่ม "เปิดการแจ้งเตือน" ในหัวข้อ "การแจ้งเตือนของ Chrome"
  2. เมื่อเบราว์เซอร์ถาม ให้กด "อนุญาต"
  3. ปุ่มจะเปลี่ยนเป็น "เปิดอยู่แล้ว ✓" สีเขียว
หากแจ้งเตือนถูกบล็อก: ไปที่ Chrome Settings → Privacy → Notifications → ค้นหาเว็บไซต์ → เปลี่ยนเป็น "Allow" แล้วรีโหลดหน้า
ตั้งค่านักเรียนที่ติดตาม
  1. เลือก บัญชีผู้ใช้ ที่ต้องการตั้งค่า (Admin ตั้งค่าให้ user อื่นได้)
  2. เปิดสวิตช์ "ติดตาม" ข้างชื่อนักเรียนที่ต้องการ
  3. ระบบบันทึกทันที — ผู้ใช้นั้นจะรับแจ้งเตือนเมื่อนักเรียนสแกน QR
ไอคอน 🔔 บนแถบนำทาง
  • ไอคอน 🔔 ที่มุมบนขวาของทุกหน้า (สำหรับ Admin/ครูผู้ช่วย) จะแสดง ตัวเลขนักเรียน ที่สแกน
  • กดไอคอน 🔔 เพื่อล้างตัวเลข
📝

8. บันทึกพัฒนาการ

notes.html — ครูเขียนบันทึก / ผู้ปกครองอ่านบันทึกของบุตรหลาน

Admin ครูผู้ช่วย ดูอย่างเดียว (อ่านอย่างเดียว)
Admin / ครูผู้ช่วย — เขียนบันทึก
  1. เลือก นักเรียน จาก Dropdown
  2. พิมพ์บันทึก เช่น ความก้าวหน้า สิ่งที่ต้องปรับปรุง หรือข้อสังเกต
  3. กด Ctrl+Enter หรือกดปุ่ม "บันทึก"
  4. บันทึกจะปรากฏในรายการประวัติทันที (Realtime)
สามารถเขียนบันทึกได้โดยตรงจาก Dashboard โดยกดปุ่ม "✏️ บันทึก" บนการ์ดนักเรียน โดยไม่ต้องเปิดหน้าใหม่
กรองบันทึก (Admin / ครูผู้ช่วย)
  • เลือกชื่อนักเรียนจาก Dropdown "กรองนักเรียน" เพื่อดูบันทึกเฉพาะคน
  • เลือก "ทุกนักเรียน" เพื่อดูบันทึกทั้งหมด
ผู้ปกครอง (ดูอย่างเดียว) — อ่านบันทึก
  • เห็นเฉพาะบันทึกของนักเรียนที่ ตั้งค่าติดตาม ไว้ (Admin ตั้งให้)
  • บันทึกใหม่จะปรากฏโดยอัตโนมัติโดยไม่ต้องรีโหลด
  • หากยังไม่มีนักเรียนที่ติดตาม ระบบจะแสดงลิงก์ไปหน้า "การแจ้งเตือน" ให้ Admin ช่วยตั้งค่า
หมายเหตุ: ผู้ปกครองไม่สามารถตั้งค่าการติดตามเองได้ ต้องให้ Admin เป็นผู้ตั้งค่าให้ผ่านหน้า "การแจ้งเตือน"
⚙️

9. ตั้งค่าระบบ

settings.html — เปลี่ยนภาษาและจัดการบัญชีผู้ใช้

Admin เท่านั้น
เปลี่ยนภาษา
  1. ไปที่หัวข้อ "ภาษา / Language"
  2. กดปุ่ม "ไทย" หรือ "English"
  3. ทุกหน้าจะเปลี่ยนภาษาทันที — ระบบจำการตั้งค่าในเบราว์เซอร์นั้น
สร้างบัญชีผู้ใช้ใหม่
  1. กดปุ่ม "+ สร้างผู้ใช้ใหม่"
  2. กรอก อีเมล และ รหัสผ่าน (อย่างน้อย 6 ตัวอักษร)
  3. กด "ยืนยันรหัสผ่าน" อีกครั้ง
  4. กดปุ่ม "สร้างบัญชี"
⚠️ หลังสร้างบัญชี: หาก Supabase ปิด Email Confirmation ระบบจะ Sign Out อัตโนมัติ Admin ต้องเข้าสู่ระบบใหม่อีกครั้ง
บทบาทเริ่มต้น: บัญชีใหม่ได้รับสิทธิ์ Admin โดยอัตโนมัติ Admin สามารถเปลี่ยนบทบาทได้ใน Supabase Dashboard
ดูรายชื่อผู้ใช้ทั้งหมด
  • ตารางด้านล่างแสดงอีเมล บทบาท และวันที่สร้างของทุก account
  • บัญชีตัวเองจะมีป้าย "คุณ" กำกับ
📋 ขั้นตอนเพิ่มผู้ปกครอง (View User)
  1. Admin สร้างบัญชีให้ผู้ปกครองในหน้า ตั้งค่าระบบ
  2. ไปที่ Supabase Dashboard → Table Editor → profiles เปลี่ยน role จาก admin เป็น view
  3. Admin ไปที่หน้า การแจ้งเตือน → เลือก account ผู้ปกครอง → เปิด toggle นักเรียนที่ต้องการ
  4. แจ้ง email/password ให้ผู้ปกครอง เข้าสู่ระบบด้วย บทบาทดูอย่างเดียว
🛠️

เข้าสู่ระบบไม่ได้ อีเมล/รหัสผ่านผิด หรือยังไม่ยืนยันอีเมล ตรวจสอบอีเมล ถ้ายังไม่ยืนยัน ให้คลิกลิงก์ในอีเมล
กล้องไม่ทำงาน เบราว์เซอร์ไม่ได้รับสิทธิ์กล้อง คลิกไอคอนกล้องในแถบ URL → อนุญาต → รีโหลดหน้า
สแกน QR แล้วไม่มีผล นักเรียนไม่ได้ลงทะเบียนวิชานั้น หรือ QR ผิดวิชา ตรวจสอบว่า QR ตรงกับวิชาที่นักเรียนลงทะเบียน
ไม่รับการแจ้งเตือน เบราว์เซอร์บล็อกการแจ้งเตือน หรือยังไม่ได้ตั้งค่า ไปหน้า "การแจ้งเตือน" → กด "เปิดการแจ้งเตือน" → อนุญาต
ข้อมูลไม่อัปเดต การเชื่อมต่อ Realtime ขาด รีโหลดหน้า (F5 หรือ Ctrl+R)
หน้า Dashboard โหลดช้า ข้อมูลนักเรียนมาก หรืออินเทอร์เน็ตช้า รอสักครู่ หรือตรวจสอบสัญญาณอินเทอร์เน็ต
ผู้ปกครองไม่เห็นบันทึก ยังไม่ได้ Map นักเรียนให้ account นั้น Admin ไปหน้า "การแจ้งเตือน" → เลือก account → เปิด toggle
ถูก redirect ไป Dashboard สิทธิ์ไม่เพียงพอสำหรับหน้านั้น ติดต่อ Admin เพื่อขอสิทธิ์ที่ถูกต้อง
